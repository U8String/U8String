# Overloads Policy
- `byte`
- `Rune`
- `U8String`
- `ReadOnlySpan<byte>`
- `ReadOnlySpan<char>` / `string`

# Validation Policy
- `ReadOnlySpan<byte>`: Always
- `ReadOnlySpan<char>`: Never
- `string`: Never
- `byte`: Always
- `Rune`: Never
- `U8String`: Conditional, on manipulation
